# 🧠 Diabetes Prediction using Machine Learning

## 📌 Overview

This project uses machine learning to predict whether a person is diabetic based on medical diagnostic features. It demonstrates the complete ML workflow — from data preprocessing to model prediction.

---

## 🚀 Key Highlights

* Cleaned and preprocessed real-world medical dataset
* Performed Exploratory Data Analysis (EDA)
* Trained a machine learning model using Scikit-learn
* Built a simple prediction system
* Demonstrates end-to-end ML pipeline

---

## 🛠 Tech Stack

* **Programming Language:** Python
* **Libraries:** Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn
* **Environment:** Jupyter Notebook / Google Colab

---

## 📊 Dataset Description

The dataset consists of medical attributes used to predict diabetes:

* Pregnancies
* Glucose Level
* Blood Pressure
* Skin Thickness
* Insulin
* BMI (Body Mass Index)
* Diabetes Pedigree Function
* Age
* Outcome (0 = Non-Diabetic, 1 = Diabetic)

---

## ⚙️ Project Workflow

1. **Data Loading**
   Imported dataset and inspected structure

2. **Data Preprocessing**

   * Handled missing/zero values
   * Cleaned inconsistent data

3. **Exploratory Data Analysis (EDA)**

   * Correlation heatmap
   * Feature relationships

4. **Model Building**

   * Split dataset into training and testing sets
   * Trained model using Scikit-learn

5. **Prediction System**

   * Built a simple function to predict diabetes
   * Tested with sample input values

---

## 📈 Results

The model is able to classify whether a person is diabetic based on input features. This project focuses on demonstrating the machine learning workflow rather than achieving maximum accuracy.
